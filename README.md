# Daybreak Browser (破晓浏览器)

**Created by:** WhiteDragon-303 (Daybreak)
**Version:** 1.0.0

A Chrome-style desktop browser built with Electron.

## Features
- Multi-tab browsing
- Bookmarks manager
- History tracking
- Password manager with auto-fill
- Dark and light themes
- Language switching (Chinese / English)
- Zoom controls with modal
- About section

## How to Run
1. Install Node.js
2. Clone this repository
3. Run `npm install`
4. Run `npm start`
