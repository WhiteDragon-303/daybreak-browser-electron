# Daybreak Browser (破晓浏览器)

**Created by:** WhiteDragon-303 (Daybreak)  
**Version:** 1.0.0

A Chrome-style desktop browser built with Electron and vanilla JavaScript.

---

## Features

- **Multi-tab browsing** with drag-to-reorder
- **Bookmarks manager** with collapsible home page bar
- **History tracking** with persistent storage
- **Password manager** with AES-256-GCM encryption
- **Dark and light themes**
- **Language switching** (Chinese / English)
- **Zoom controls** with modal and keyboard shortcuts
- **Ad & tracker blocking** via Ghostery adblocker
- **Custom search engines** - add/delete any search engine
- **Multiple built-in engines** - Google, Baidu, Bing, DuckDuckGo, Sogou
- **Right-click context menus** on web pages, tabs, and browser chrome
- **Window state persistence** - remembers size and position
- **Keyboard shortcuts** - Ctrl+T, Ctrl+W, Ctrl+D, Ctrl+L, F5, etc.
- **Search selected text** via right-click context menu
- **View page source** and **Inspect element**
